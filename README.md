# TechSphereX Website

A modern IT company website built with Vite + React + SCSS.

## Color Theme
Extracted from the TechSphereX logo:
- Primary (Cyan): #00B4D8
- Deep Blue: #0077B6
- Light Cyan: #48CAE4
- Dark Navy BG: #0A0F1E

## Setup

```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Pages
- `/` - Home (hero, services, expertise, products, clients, contact)
- `/about` - About (mission, vision, core values, story)
- `/services` - Services (9 service cards)
- `/products` - Products (NexoEats, Rewardiz, SphereCart)
- `/careers` - Careers (job listings)
- `/contact` - Contact (form + info)
